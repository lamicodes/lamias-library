\# Lamia's Library



A Ghibli-inspired portfolio site — painterly, cozy, and a little magical.



\## About



Lamia's Library is a personal portfolio designed as a series of rooms you can wander through — a lobby, a study, an atrium, and a letters room — each with its own atmosphere and purpose. Built with hand-painted backgrounds and a love for slow, deliberate web design.



\## Tech Stack



\- HTML / CSS / JavaScript — vanilla, no frameworks

\- GitHub Pages — hosting

\- Namecheap — custom domain DNS

\- `<picture>` elements — responsive art direction for mobile vs. desktop backgrounds



\## Rooms



\- Lobby — the entrance

\- Study — projects and work

\- Atrium — about me

\- Letters — contact form



\## Status



Currently polishing mobile responsiveness. Desktop is solid; mobile portrait versions of each room background are in place (`lobby-mobile.png`, `study-mobile.png`, `atrium-mobile.png`, `letters-mobile.png`). Working through a few positioning quirks on smaller screens.
